# HStack

HStack is a convenience wrapper around Stack with a fixed horizontal (row) direction. Use it for horizontal layouts.

## Import

```tsx
import { HStack } from "@tosui/react";
```

## Common Patterns

### Navigation Bar

```tsx
<HStack as="nav" gap={6} align="center" justify="space-between" p={4}>
  <Box fontWeight="bold">Logo</Box>
  <HStack gap={4}>
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
  </HStack>
</HStack>
```

### Button Group

```tsx
<HStack gap={2}>
  <Button variant="solid">Save</Button>
  <Button variant="outline">Cancel</Button>
</HStack>
```

## Props Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| gap | `0-32` | - | Gap between children (spacing multiplier) |
| align | `"flex-start"` \| `"center"` \| `"flex-end"` \| `"stretch"` \| `"baseline"` | - | Cross-axis alignment |
| justify | `"flex-start"` \| `"center"` \| `"flex-end"` \| `"space-between"` \| `"space-around"` \| `"space-evenly"` | - | Main-axis alignment |
| wrap | `boolean` | `false` | Enable flex wrapping |
| as | `ElementType` | `"div"` | HTML element to render |

HStack accepts all [Stack](/docs/components/stack) props except `direction`.

# VStack

VStack is a convenience wrapper around Stack with a fixed vertical (column) direction. Use it for vertical layouts.

## Import

```tsx
import { VStack } from "@tosui/react";
```

## Common Patterns

### Form Layout

```tsx
<VStack gap={4} align="stretch">
  <Input placeholder="Name" />
  <Input placeholder="Email" />
  <Textarea placeholder="Message" />
  <Button>Submit</Button>
</VStack>
```

### Card Content

```tsx
<Card>
  <CardBody>
    <VStack gap={3} align="flex-start">
      <Heading fontSize="xl">Card Title</Heading>
      <Text color="foreground-muted">Card description goes here.</Text>
      <Button size="sm">Learn More</Button>
    </VStack>
  </CardBody>
</Card>
```

## Props Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| gap | `0-32` | - | Gap between children (spacing multiplier) |
| align | `"flex-start"` \| `"center"` \| `"flex-end"` \| `"stretch"` \| `"baseline"` | - | Cross-axis alignment |
| justify | `"flex-start"` \| `"center"` \| `"flex-end"` \| `"space-between"` \| `"space-around"` \| `"space-evenly"` | - | Main-axis alignment |
| wrap | `boolean` | `false` | Enable flex wrapping |
| as | `ElementType` | `"div"` | HTML element to render |

VStack accepts all [Stack](/docs/components/stack) props except `direction`.

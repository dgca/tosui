# Flex

Flex is an explicit flexbox container with shorthand props for common flexbox properties. It provides more granular control than Stack.

## Import

```tsx
import { Flex } from "@tosui/react";
```

## Basic Usage

## Flex Item Properties

Flex supports `grow`, `shrink`, and `basis` for controlling the flex container itself.

```tsx
<Flex grow={1} shrink={0} basis="200px">
  This flex container will grow to fill space
</Flex>
```

## Props Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| direction | `"row"` \| `"column"` \| `"row-reverse"` \| `"column-reverse"` | `"row"` | Flex direction |
| wrap | `"nowrap"` \| `"wrap"` \| `"wrap-reverse"` | `"nowrap"` | Flex wrap behavior |
| justify | `"flex-start"` \| `"center"` \| `"flex-end"` \| `"space-between"` \| `"space-around"` \| `"space-evenly"` | - | Main-axis alignment |
| align | `"flex-start"` \| `"center"` \| `"flex-end"` \| `"stretch"` \| `"baseline"` | - | Cross-axis alignment |
| alignContent | `"flex-start"` \| `"center"` \| `"flex-end"` \| `"space-between"` \| `"space-around"` \| `"space-evenly"` | - | Multi-line alignment |
| gap | `0-32` | - | Gap between children |
| gapRow | `0-32` | - | Row gap |
| gapColumn | `0-32` | - | Column gap |
| basis | `string` | - | Flex basis |
| grow | `number` | - | Flex grow |
| shrink | `number` | - | Flex shrink |
| alignSelf | `"auto"` \| `"flex-start"` \| `"center"` \| `"flex-end"` \| `"stretch"` \| `"baseline"` | - | Self alignment in parent flex |
| as | `ElementType` | `"div"` | HTML element to render |
